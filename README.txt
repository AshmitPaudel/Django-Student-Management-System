# Student Management System

## Overview
This is a Django-based Student Management System designed to manage student information, including adding, updating, and deleting student records. The project also includes a REST API for interacting with student data.

## Project Structure
- `StudentSystem/` - Main Django project directory containing all the code.
- `StudentSystem_env/` - Virtual environment directory (ensure to activate it before running the project).
- `requirements.txt` - File listing all the dependencies required for the project.
